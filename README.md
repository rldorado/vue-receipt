# Vue Receipt

Small webapp made for generate simple receipts. It makes use of `jspdf` & `jspdf-autotable` to generate final PDFs.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
